# Screenshot preparation

Add these files before running the final release tests:

```text
docs/images/panel-overview.jpg
docs/images/panel-expanded-tree.jpg
docs/images/panel-search.jpg
```

Use neutral entities and remove or blur personal names, addresses, precise locations, tokens, private URLs, and sensitive alarm, lock, presence, camera, or health-related entities.

`panel-overview.jpg` should show the sidebar entry, title, root, at least one parent and child, and relation arrows.

`panel-expanded-tree.jpg` should show at least two expanded levels with parents above and children below the root.

`panel-search.jpg` should show a useful query, multiple results, the total-result summary, and a selected result.
